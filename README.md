# C# Number Guessing Game

This project is a beginner-friendly C# console game where the computer randomly generates a number and the user tries to guess it.

The program provides feedback if the guess is too high or too low until the correct number is guessed.

## Features

- Random number generation
- User guess input
- Feedback (Too high / Too low / Correct)
- Input validation
- Replay option

## Concepts Practiced

- Random class
- Variables and data types
- Conditional statements (if / else)
- While loops
- User input with Console.ReadLine()
- Input validation with TryParse

## How to Run

1. Clone the repository
2. Open the project in Visual Studio or VS Code
3. Run the program
